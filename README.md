# Translate app written in Rust ( Works only in linux )
__**ray_translate**__ - quick acess translator(obviously if you use shortcut).

# Installation

## Archlinux
```
yay -S ray_translate
```

## Other distros
install translate-shell and build repo, then set shortcut etc.
